    Jozi JUG Dojo - White belt kata 1

    Welcome to the first white belt kata!

    In this kata we will compile and run a simple program that
    1) Prints out the current time
    2) Adds 10 minutes to the current time
    3) Demonstrates the concept of immutabliliy in Java
    4) Prints the time then minutes after the current time

    Desired outcome

    Don't worry to much about the content of the program. The goal is to successfully compile and run this program on the supplied virtual machine.

    What you will need

    A computer, VirtualBox Open Source edition , The Jozi JUG Dojo White Belt virtual machine. Please join the Kata Forums and join the community

    The code that you will compile and run. Don't worry to much about the code as we will cover that in the next kata.

    package za.co.jozijugdojo.whitebelt1.hellojava8;

import java.time.LocalDateTime;
import java.time.temporal.ChronoUnit;

public class App {

    public static void main(String[] args) {
        LocalDateTime now = LocalDateTime.now();
        System.out.println("It is now :"+now);
        
        LocalDateTime tenMinutesFromNow = now.plus(10, ChronoUnit.MINUTES);
        
        System.out.println("Now did not change, it is immutable (will never change) :"+now);        
        System.out.println("Ten minutes from now is "+tenMinutesFromNow);
    }
}
